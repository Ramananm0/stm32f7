# Archived Maintenance Shell Scripts

These scripts are one-off checks, probes, or verification helpers that are not part of the normal rover bringup path.

Kept in the live tree:
- `rover_live/laptop/start.sh`
- `rover_live/pi/start_pi.sh`
- `rover_live/orbslam3/*.sh`
- `ros2/use_pi_fastdds.sh`
- service and install helpers such as `pi_enable_*`, `pi_fix_*`, `pi_install_*`, `pi_create_*`, and `pi_patch_*`

Archived here:
- runtime checks
- post-reboot checks
- topic probes
- camera/AI inspection scripts
